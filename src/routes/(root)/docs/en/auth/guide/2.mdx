description: Prepare for payment integration using the Merchant ID.

import Tabs from "~/components/gitbook/Tabs";
import Hint from "~/components/Hint";

On the checkout page, initialize the IMP object using your [Merchant ID](../../../../ready/3).

<Tabs>
  <Tabs.Tab title="JavaScript">
    ```ts title="Client-side"
    const IMP = window.IMP; // Can be omitted
    IMP.init("Merchant ID"); // Example: imp00000000
    ```
  </Tabs.Tab>

  <Tabs.Tab title="JavaScript (ES Next)">
    ```ts title="Client-side"
    const IMP = window.IMP; // Can be omitted
    IMP.init("{Merchant ID}"); // Example: imp00000000a
    ```
  </Tabs.Tab>
</Tabs>

<Hint style="warning">
  Be sure to initialize the IMP object **only once**.
</Hint>

**Calling a sub-merchant (Tier) payment window**

<Hint style="info">
  `IMP.agency('Merchant ID', 'ABC');  //`<mark style="color:red;">**ABC = Tier Code**</mark>
</Hint>
