1. [HYPHEN PAY & 바로계좌결제 운영 백오피스](http://payadmin.hyphen.im/) 접속 후 **바로계좌결제** 선택 후
   로그인을 합니다.

   <Figure src={image1} caption="HYPHEN 백오피스 로그인 화면" />

2. \[상점관리]→\[바로계좌결제 상점정보관리]를 클릭합니다.

3. 상점명을 클릭합니다.

   <Figure src={image2} caption="HYPHEN [상점관리] - [상점정보관리] 예시 화면" />

4. \[상점 기본정보]에서 \[상점번호]를 확인합니다.

   <Figure src={image3} caption="HYPHEN [상점 기본정보] 예시 화면" />

5. \[암호화 키, 토큰]에서 \[암호화 키]와 \[암호화 토큰]을 확인합니다.

   <Figure src={image4} caption="HYPHEN [암호화 키, 토큰] 예시 화면" />

6. 포트원 콘솔에서 채널 추가 시 \[PG상점아이디(상점 번호)], \[암호화 키], \[암호화 토큰]을
   입력한 후 \[저장]을 클릭합니다.
