### Deletes a billing key. You must check for any scheduled payments on the billing key.

<Swagger method="delete" path="/subscribe/customers/{customer_uid}" baseUrl="https://api.iamport.kr" summary=" Deletes the specified billing key.">
<SwaggerDescription>
A deleted billing key cannot be restored.

</SwaggerDescription>

### Parameters

#### Path

<SwaggerParameter name="customer_uid" type="String" required="true">
<mark style="color:red;">

**Billing key**

</mark>

</SwaggerParameter>

#### Query

<SwaggerParameter name="reason" type="String">
**Reason for deletion**

</SwaggerParameter>
<SwaggerParameter name="extra[requester]" type="String">
**Requester**

(for Naver Pay only)

</SwaggerParameter>
