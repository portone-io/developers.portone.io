### Cancels a scheduled payment before execution.

<Swagger method="post" path="/subscribe/payments/unschedule" baseUrl="https://api.iamport.kr" summary="Cancels a scheduled payment before execution.">
<SwaggerDescription>


</SwaggerDescription>

### Parameters

#### Body

<SwaggerParameter required="true" name="customer_uid" type="String">
<mark style="color:red;">

**Billing key**

</mark>

</SwaggerParameter>
<SwaggerParameter name="merchant_uid" type="Array">
**Order ID**

If omitted, cancels all scheduled payments associated with customer\_uid.

</SwaggerParameter>
