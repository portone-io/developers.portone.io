## Gets Benepia points using user's Benepia account ID and password through KCP.

<Swagger method="post" path="/benepia/point" baseUrl="https://api.iamport.kr" summary="Gets Benepia points using user's Benepia account ID and password through KCP.">
  <SwaggerDescription>
    Requires KCP site code.
  </SwaggerDescription>

  ### Parameters

  #### Body

  <SwaggerParameter name="benepia_user" type="String" required="true">
    <mark style="color:red;">
      **Benepia account ID**
    </mark>
  </SwaggerParameter>

  <SwaggerParameter name="benepia_password" type="String" required="true">
    <mark style="color:red;">
      **Benepia account password**
    </mark>
  </SwaggerParameter>

  <SwaggerParameter name="pg" type="String">
    **PG code**

    If there are multiple KCP PG settings, specify as **kcp.\{KCP site code}**. If omitted, the KCP pg setting in Admin console is used.
  </SwaggerParameter>

  ### Responses

  <SwaggerResponse status="200: OK" description="Success">
    <Tabs>
      <Tab title="BenepiaPointResponse">
        **`code`** <mark style="color:red;">**\***</mark> <mark style="color:purple;">**integer**</mark>

        **Response code**

        0: success, Not 0: check the message

        **`message`** <mark style="color:red;">**\***</mark> <mark style="color:green;">**string**</mark>

        **Response message**

        A non-zero code includes a message like 'Invalid payment info'.

        **`response`` `**<mark style="color:red;">**`(BenepiaPointAnnotation, optional)`**</mark>
      </Tab>
    </Tabs>

    <Tabs>
      <Tab title="BenepiaPointAnnotation">
        **`point`**<mark style="color:red;">**`*`**</mark><mark style="color:orange;">**`integer`**</mark>

        **`Benepia points`**
      </Tab>
    </Tabs>
  </SwaggerResponse>
