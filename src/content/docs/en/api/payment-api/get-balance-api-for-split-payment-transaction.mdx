## Gets the balance of a transaction paid for using multiple payment methods.

<Swagger method="get" path="/payments/{imp_uid}/balance" baseUrl="https://api.iamport.kr" summary="Gets the balance of a transaction paid for using multiple payment methods.">
  <SwaggerDescription>
    This is currently supported only by PAYCO.
  </SwaggerDescription>

  ### Parameters

  #### Path

  <SwaggerParameter name="imp_uid" type="String" required="true">
    <mark style="color:red;">
      **i'mport order ID**
    </mark>
  </SwaggerParameter>

  ### Responses

  <SwaggerResponse status="200: OK" description="Success">
    <Tabs>
      <Tab title="PaymentBalanceResponse">
        **`code`** <mark style="color:red;">**\***</mark> <mark style="color:purple;">**integer**</mark>

        **Response code**

        0: success, Not 0: check the message

        **`message`** <mark style="color:red;">**\***</mark> <mark style="color:green;">**string**</mark>

        **Response message**

        A non-zero code includes a message like 'Invalid payment info'.

        **`response`` `**<mark style="color:red;">**`(PaymentBalanceResponseAnnotation, optional)`**</mark>
      </Tab>
    </Tabs>

    <Tabs>
      <Tab title="PaymentBalanceResponseAnnotation">
        **`amount`` `**<mark style="color:purple;">**`Integer`**</mark>

        **`Total amount`**

        **`cash_receipt`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`Cash receipt amount`**

        **`primary`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`Payment method 1`**(credit card, account transfer, virtual account, mobile micropayment) amount

        **`secondary`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`Payment method 2`** PG/credit card points amount

        **`discount`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`PG/credit card discount amount`**

        **`histories`` `**<mark style="color:red;">**`(Array[PaymentBalanceBaseAnnotation], optional)`**</mark>

        **`PaymentBalance history`**
      </Tab>
    </Tabs>

    <Tabs>
      <Tab title="PaymentBalanceAnnotation">
        **`tax_free`` `**<mark style="color:purple;">**`Integer`**</mark>

        **`Tax free amount`** (excludes any refunded amount)

        **`supply`` `**<mark style="color:purple;">**`Integer`**</mark>

        **`Taxed amount`** (excludes any refunded amount)

        **`vat`** <mark style="color:purple;">**`Integer`**</mark>

        **`Tax`** (excludes any refunded amount)

        **`service`` `**<mark style="color:purple;">**`Integer`**</mark>

        **`Fee`** (excludes any refunded amount)
      </Tab>
    </Tabs>

    <Tabs>
      <Tab title="PaymentBalanceBaseAnnotation">
        **`cash_receipt`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`Cash receipt amount`**

        **`primary`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`Payment method 1`**(credit card, account transfer, virtual account, mobile micropayment) amount

        **`secondary`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`Payment method 2`** PG/credit card points amount

        **`discount`` `**<mark style="color:red;">**`(PaymentBalanceAnnotation):`**</mark>

        **`PG/credit card discount amount`**

        **`created  *`` `**<mark style="color:purple;">**`integer`**</mark>

        **`Balance created at (UNIX timestamp)`**
      </Tab>
    </Tabs>
  </SwaggerResponse>

  <SwaggerResponse status="401: Unauthorized" description="Missing or invalid access token">
