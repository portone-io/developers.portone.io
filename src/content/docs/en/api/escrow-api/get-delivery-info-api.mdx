## Gets delivery information for an escrow payment transaction.

<Details>
  <p slot="summary"><strong>Supported PGs</strong></p>

  - KG INICIS
  - NHN KCP
  - PAYJOA (Daou)
</Details>

<Swagger method="get" path="/escrows/logis/{imp_uid}" baseUrl="https://api.iamport.kr" summary="Gets delivery information for an escrow payment transaction.">
  <SwaggerDescription />

  ### Parameters

  #### Path

  <SwaggerParameter name="imp_uid" type="String" required="true">
    <mark style="color:red;">
      **i'mport transaction ID**
    </mark>
  </SwaggerParameter>

  ### Responses

  <SwaggerResponse status="200: OK" description="Success">
    <Tabs>
      <Tab title="EscrowLogisResponse">
        **`code`** <mark style="color:red;">**\***</mark> <mark style="color:purple;">**integer**</mark>

        **Response code**

        0: success, Not 0: check the message

        **`message`** <mark style="color:red;">**\***</mark> <mark style="color:green;">**string**</mark>

        **Response message**

        A non-zero code includes a message like 'Invalid payment info'.

        **`response`**<mark style="color:red;">**`(EscrowLogisAnnotation, optional)`**</mark>
      </Tab>
    </Tabs>

    <Tabs>
      <Tab title="EscrowLogisAnnotation">

        **`Tracking number`**

        **`sent_at`**<mark style="color:red;">**`*`**</mark><mark style="color:orange;">**`integer`**</mark>
