### Cancels a virtual account before payment is deposited.

<Details>
<p slot="summary"><strong>Supported PGs</strong></p>

* **KG INCIS**
* **NHN KCP**
* **TOSS Payments**
* **Nice Payments**
* **KICC**
* **Settle Bank**
* **Smartro**

</Details>

<Swagger method="delete" path="/vbanks/{imp_uid}" baseUrl="https://api.iamport.kr" summary="Cancels a virtual account before payment is deposited.">
<SwaggerDescription>
You can cancel a virtual account before deposit to prevent the buyer from depositing by mistake. 

**imp_uid**

 must be specified. (Since there can be multiple virtual accounts pending payment deposit for a single merchant_uid, you can only cancel a virtual account based on the imp_uid)

</SwaggerDescription>

### Key request parameter description

> **`pg_api_key`` `**<mark style="color:green;">**`String`**</mark>
>
> **API Key**
>
> This is the API Key from the KG INICIS Merchant console and is used for virtual account issuance and cancellation. A hashData mismatch error occurs if you specify a missing or invalid key.
>
> (**Required for KG INICIS only**)
