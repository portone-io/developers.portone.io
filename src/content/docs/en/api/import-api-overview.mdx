## Learn how to configure HTTP header in i'mport API call.

1. Call [**/users/getToken**](rest-api-access-token) with **API Key** & **API secret**
   to get an <mark style="color:red;">**`access_token`**</mark>.

2. In the API call, include the <mark style="color:red;">**`access_token`**</mark> as **`Authorization:`**<mark style="color:red;">**`access_token`**</mark>
   or **`X-ImpTokenHeader:`**<mark style="color:red;">**`access_token`**</mark> in the HTTP header.

<Hint style="info">
  **Content-Type**

  API Content-Type supported in i'mport is:

  **Application.json;charset=UTF-8**
</Hint>

<Hint style="warning">
  **Agency Merchant**

  Sub-merchants must specify the following in the **`HTTP Header`** and use the **Agent account's** <mark style="color:blue;">**`API key`**</mark>
  and <mark style="color:blue;">**`Secret`**</mark> to call the i'mport API.

  - **`"Tier" : "`**<mark style="color:red;">**`Tier code`**</mark>(Sub-merchant Tier Code)"
</Hint>
