## Updates the status of a PAYCO order.

<Swagger method="post" path="/payco/orders/status/{imp_uid}" baseUrl="https://api.iamport.kr" summary="Updates the status of a PAYCO order.">
  <SwaggerDescription />

  ### Parameters

  #### Path

  <SwaggerParameter name="imp_uid" type="String" required="true">
    <mark style="color:red;">
      **i'mport transaction ID**
    </mark>
  </SwaggerParameter>

  #### Body

  <SwaggerParameter name="status" type="String" required="true">
    <mark style="color:red;"> **Status code** </mark>

    `DELIVERY_START`

    `PURCHASE_DECISION`

    `CANCELED`
  </SwaggerParameter>

  ### Responses

  <SwaggerResponse status="200: OK" description="Success">
    <Tabs>
      <Tab title="PaycoStatusResponse">
        **`code`** <mark style="color:red;"> **\*** </mark> <mark style="color:purple;"> **integer** </mark>

        **Response code**

        0: success, Not 0: check the message

        **`message`** <mark style="color:red;"> **\*** </mark> <mark style="color:green;"> **string** </mark>

        **Response message**

        A non-zero code includes a message like 'Invalid payment info'.

        **`response`** <mark style="color:red;"> **(PaycoStatusAnnotation, optional)** </mark>
      </Tab>
    </Tabs>

    <Tabs>
      <Tab title="PaycoStatusAnnotation">
        **`status`** <mark style="color:red;"> **`*`** </mark><mark style="color:green;"> **`string`** </mark>

        **`PAYCO updated order status`**

        \['DELIVERY\_START', 'PURCHASE\_DECISION', 'CANCELED']
      </Tab>
    </Tabs>
  </SwaggerResponse>

  <SwaggerResponse status="401: Unauthorized" description="Missing or invalid token">
    ```ts
    {
      // Response
    }
    ```
  </SwaggerResponse>
</Swagger>

<Details>
  <p slot="summary">Response Model Schema</p>

  ```json
  {

    "message": "string",
    "response": {
      "status": "DELIVERY_START"
    }
  }
  ```
