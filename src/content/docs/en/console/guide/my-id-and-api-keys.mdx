## My ID & API keys

## Where to find ID & API keys

![](</gitbook-assets/en/Untitled (8).png>)

Click `My ID and API Keys` at the top right of the `Merchant account` page to view and edit the merchant's ID and API information.

## Merchant ID

![](</gitbook-assets/en/Untitled (9).png>)

Merchant ID is automatically issued when you sign up from the Admin page and is used to identify the merchant when using the i'mport module.

[What is Merchant ID?](../../ready/3.)

## Rest API Key & API Secret Key

[How do you use REST API and Secret keys?](../../ready/3.)

To request for a reissuance of the API Secret key, click `Reissue`. Since this invalidates any existing Secret key, check for one in advance.

## Tier code

When setting up a sub-merchant, specify a tier code and use it to call `IMP.agency('Merchant ID', 'tier code')` instead of `IMP.init`.\
