## Authenticated payment

<Tabs>
<Tab title="Test">
### **Test environment setup**

1. [i'mport Admin console](https://admin.iamport.kr)→ **Payment integration** → **Test settings** → Select PG (**KICC**) → **KICC** -> **Add**
2. Check the auto-generated test MID value and click **Save**

<Figure src="/gitbook-assets/en/image (270).png" caption="Payment integration → Test settings → Select PG (KICC) → KICC -> Add" />

<Figure src="/gitbook-assets/en/image (251).png" caption="Check the auto-generated test MID value and click Save " />

</Tab>

<Tab title="Live">
### Live environment setup

1. **Payment integration -> Live settings**
2. Select PG (**KICC**) → **KICC** -> **Add**
3. Enter merchant info issued by KICC upon approval by the credit card company.

<Figure src="/gitbook-assets/en/image (192).png" caption="Payment integration -> Live settings" />

<Figure src="/gitbook-assets/en/image (205).png" caption="Select PG (KICC) → KICC -> Add" />

<Figure src="/gitbook-assets/en/image (222).png" caption="Enter merchant info issued by KICC upon approval by the credit card company" />

</Tab>
</Tabs>



## Non-authenticated payment

<Tabs>
<Tab title="Payment window">
### Test environment setup

1. [i'mport Admin console](https://admin.iamport.kr)→ **Payment integration** → **Test settings** → Select PG (**KICC**) → **KICC** -> **Add**
2. Check the auto-generated MID value and then click **Save**.

<Figure src="/gitbook-assets/en/image (188).png" caption="Payment integration → Test settings → Select PG (KICC) → KICC -> Add" />

<Figure src="/gitbook-assets/en/image (200).png" caption="Check the auto-generated MID value and then click Save" />

### Live environment setup

1. **Payment integration -> Live settings**
2. **KICC -> KICC -> Add**
3. Enter **MID** issued at contract signing -> **Save**

<Figure src="/gitbook-assets/en/image (183).png" caption="Payment integration -> Live settings" />

<Figure src="/gitbook-assets/en/image (220).png" caption="KICC -> KICC -> Add" />

<Figure src="/gitbook-assets/en/image (207).png" caption="Enter MID issued at contract signing -> Save" />

</Tab>

**Not supported.**

</Tab>
</Tabs>
