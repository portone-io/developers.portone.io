## Authenticated payment

<Tabs>
  <Tab title="Test">
    ### **Test environment setup**

    1. [i'mport Admin console](https://admin.iamport.kr)→ **Payment integration** → **Test settings** → Select PG (**NICE Payments**) → **NICE Payments** -> **Add**
    2. Check the auto-generated test MID value and click **Save**

    > A payment is made on the i'mport test MID account.

    <Figure src="/gitbook-assets/en/image (119).png" caption="Payment integration → Test settings → Select PG (NICE Payments) → NICE Payments -> Add" />

    <Figure src="/gitbook-assets/en/image (128).png" caption="Check the auto-generated MID value and click Save" />

    In NICE Payments test mode, the following payment methods are **not supported**.

    - Credit card: NH card, KB Kookmin card
    - Mobile micropayments: KT

    <Hint style="info">
      **Payments made in NICE Payments test mode are cancelled daily between 23:00\~23:50.**

      Since cancelled transactions are not automatically reflected in i'mport, it is recommended that you submit a cancellation request after testing.
    </Hint>
  </Tab>

  <Tab title="Live">
    ### Live environment setup

    1. **Payment integration -> Live settings**
    2. Select PG (**NICE Payments**) → **NICE Payments** -> **Add**
    3. Enter merchant info issued by NICE Payments upon approval by the credit card company.

    <Figure src="/gitbook-assets/en/image (218).png" caption=" Payment integration -> Live settings" />

    <Figure src="/gitbook-assets/en/image (193).png" caption="Select PG (NICE Payments) → NICE Payments -> Add" />

    <Figure src="/gitbook-assets/en/image (214).png" caption="Enter merchant info issued by NICE Payments upon approval by the credit card company" />

    <Hint style="info">
      **Precautions when using virtual accounts**

      You must log in to the NICE Payments Admin page and set the **virtual account deposit notification URL**. (If omitted,  deposit will not be applied.)
    </Hint>
  </Tab>
</Tabs>

## Non-authenticated payment

i'mport currently **only supports API method** for NICE Payments non-authenticated payments.

<Tabs>
  <Tab title="API">
    ### **Test environment setup**

    1. [i'mport Admin console](https://admin.iamport.kr)→ **Payment integration** → **Test settings** → Select PG (**NICE Payments**) → <mark style="color:red;">**NICE Payments API**</mark> -> **Add**
    2. Check the auto-generated test MID value and click **Save**

    <Figure src="/gitbook-assets/en/image (115).png" caption="Payment integration → Test settings → Select PG (NICE Payments) → NICE Payments API -> Add" />

    <Figure src="/gitbook-assets/en/image (282).png" caption="Check the auto-generated test MID value and click Save" />

    ### Live environment setup

    - **Payment integration -> Live settings**
    - Select PG (**NICE Payments**) → <mark style="color:red;">**NICE Payments API**</mark> -> **Add**
    - Enter merchant info issued by NICE Payments upon approval by the credit card company

    <Figure src="/gitbook-assets/en/image (250).png" caption="Payment integration -> Live settings" />

    <Figure src="/gitbook-assets/en/image (172).png" caption="Select PG (NICE Payments) → NICE Payments API -> Add " />

    <Figure src="/gitbook-assets/en/image (173).png" caption="Enter merchant info issued by NICE Payments upon approval by the credit card company" />
  </Tab>
</Tabs>
