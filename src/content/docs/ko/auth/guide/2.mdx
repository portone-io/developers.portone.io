#### 주문 페이지에서 가맹점 식별코드를 이용하여 IMP 객체를 초기화 합니다.

<Hint style="warning">
해당 초기화 작업을 **1회 이상** 처리되지 않도록 유의하세요

</Hint>

<Tabs>
<Tab title="JavaScript">
```javascript title="Client-side"
  IMP.init("가맹점 식별코드"); // 예: 'imp00000000a'
```
</Tab>
</Tabs>




<Hint style="info">
**하위가맹점(Tier) 결제창 호출 방법**

하위 가맹점인 경우 IMP.init 대신 <mark style="color:red;">`IMP.agency('가맹점 식별코드', '티어코드')`</mark>를 호출하여 IMP 객체를 초기화합니다.

</Hint>

<Tabs>
<Tab title="JavaScript">
```javascript title="Client-side"
  IMP.agency('가맹점 식별코드', '티어코드'); // 예: 'imp00000000a', '123'
```
</Tab>
</Tabs>
