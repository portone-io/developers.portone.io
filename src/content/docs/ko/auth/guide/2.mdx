#### 주문 페이지에서 가맹점 식별코드를 이용하여 IMP 객체를 초기화 합니다.

<Hint style="warning">
아래 초기화 함수를 **2회 이상** 중복되게 호출하지 않도록 주의해주세요.
</Hint>

<Tabs>
<Tab title="JavaScript">
```javascript title="client-side"
IMP.init('가맹점 식별코드') // 예: 'imp00000000a'
```
</Tab>
</Tabs>

&nbsp;

#### 하위가맹점 결제창 호출 방법
하위 가맹점인 경우 `IMP.init()` 대신, `IMP.agency('가맹점 식별코드', '하위
가맹점 티어코드')`를 호출해야 합니다.

```javascript
// IMP.init() 대신 아래 함수를 사용해야함
IMP.agency('가맹점 식별코드', '티어코드') // 예: 'imp00000000a', '123'
```
