### Shopify 스토어에 PortOne 연결하기
#### 1. 먼저 Shopify 앱스토어에서 PortOne Payments 앱을 추가해 줍니다.

‘앱 추가’ 버튼을 클릭합니다.

![image](https://github.com/portone-io/developers.portone.io/assets/85064711/936bfcb5-2634-4f05-b176-7fd5735578fb)

우측 상단에 Install app 을 클릭하여 플러그인을 설치합니다.

![image](https://github.com/portone-io/developers.portone.io/assets/85064711/b8f6493f-9587-490b-bdcb-40e22bbb5151)


#### 2. Shopify 스토어의 Settings > Payments 페이지에서 PortOne Payments 앱이 제공하는 결제수단을 선택해 줍니다. 이 부분은 엔드유저에게 스토어에서 제공하는 결제수단에 대한 마크를 노출시킵니다. 실제로 제공되는 결제수단과는 무관합니다.

Visa, Mastercard 등의 결제수단 마크를 추가할 수 있습니다.

![image](https://github.com/portone-io/developers.portone.io/assets/85064711/70769ddb-68a6-4de6-9b05-e20dd6167dfe)

#### 3. Shopify 스토어의 PortOne Payments 설정페이지에서 Key 값을 추가하고, Verify 해 줍니다.

Shopify > Settings > Payments 페이지 상단에 ‘Manage’ 버튼을 클릭하여 PortOne Payments 세팅 페이지에 접속합니다. PortOne Key 값을 입력하게 되어있습니다.

![image](https://github.com/portone-io/developers.portone.io/assets/85064711/8c9e93d2-3b80-4444-8e61-6c17ea461896)

PortOne(Global) 어드민 > Management > General 화면에서 Key 값을 확인하고 복사해 줍니다.

![image](https://github.com/portone-io/developers.portone.io/assets/85064711/4382052a-377e-4156-b634-c2ebbe889a80)

Key 값을 입력한 후 ‘Verify Now’ 버튼을 클릭하여 연동상태를 확인합니다. 이제 모든 연동이 완료되었습니다!
![image](https://github.com/portone-io/developers.portone.io/assets/85064711/5ec5ff10-451b-4bb9-99de-a9d20b7ff2e1)
