### 네이버페이 주문형 상품 반품거절 처리 API

<Swagger method="post" path="/payments/{imp_uid}/naver/reject-return" baseUrl="https://api.iamport.kr" summary="네이버페이 주문형 상품 반품거절 처리">
<SwaggerDescription>

네이버페이 상품주문들을 반품거절처리합니다.

주문상품의 종류가 여러 개인 경우, 한 건의 포트원 거래건(imp_uid)에 여러 개의 네이버페이 상품주문이 존재하며 일부 또는 전체를 반품거절 처리할 수 있습니다.

`product_order_id` 가 생략될 경우, imp_uid에 해당되는 모든 네이버페이 상품주문들을 반품거절 처리합니다.

반품시 배송비 제외, 혹은 추가 청구액이 필요한 경우 반품보류 처리(POST 반품요청상품 반품보류 처리 API) 사용

여러 건의 상품주문이 반품거절 처리 시도되는 경우, 일부 주문건만 반품거절 처리에 성공할 수 있습니다.

시도된 상품주문 중 <b>반품거절 처리에 성공한 상품주문</b>에 대한 최종 정보만 제공됩니다.

(NaverProductOrderAnnotation 참조 : 응답 DATA속성은 <b>네이버페이 주문형 포트원 거래고유번호 기준 상품주문 조회 API</b> 의 data 속성과 동일한 구조를 가집니다.)

</SwaggerDescription>
