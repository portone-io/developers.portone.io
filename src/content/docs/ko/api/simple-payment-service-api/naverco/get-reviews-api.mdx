### 네이버페이 주문형 네이버페이 구매평 조회

<Swagger method="get" path="/naver/reviews" baseUrl="https://api.iamport.kr" summary="네이버페이 구매평 조회">
<SwaggerDescription>

네이버페이 구매평을 조회합니다.

</SwaggerDescription>

### Parameters

#### Path

<SwaggerParameter name="from" type="Integer" required="true">
<mark style="color:red;">

**조회기간 시작** (UNIX timestamp)

</mark>

</SwaggerParameter>

<SwaggerParameter name="to" type="Integer" required="true">
<mark style="color:red;">

**조회기간 종료** (UNIX timestamp)

</mark>

</SwaggerParameter>

<SwaggerParameter name="review_type" type="String" required="true">
<mark style="color:red;">

**구매평 유형** 

</mark>

- general : 일반 구매평
- preminum : 프리미엄 구매평
</SwaggerParameter>

### Responses

<SwaggerResponse status="200: OK" description="구매평 조회 성공">
<Tabs>
<Tab title="NaverReviewsResponse">
**`code`** <mark style="color:red;">**\***</mark> <mark style="color:purple;">**integer**</mark>

**응답코드**

**응답메세지**

</Tab>
