#### **주의사항**

해당 API 는 카카오페이 정책이슈로 인해 **2022년 하반기에 종료**될 예정입니다.

</Hint>

### [카카오페이 주문내조회 API](https://developers.kakao.com/docs/restapi/kakaopay-api#%EC%A3%BC%EB%AC%B8%EB%82%B4%EC%97%AD%EC%A1%B0%ED%9A%8C)를 래핑한 API 입니다.

<Swagger method="get" path="/kakao/payment/orders" baseUrl="https://api.iamport.kr" summary="주문내역 조회 API">
<SwaggerDescription>
지원되는 request가 유사하며 응답되는 response는 카카오페이 API 와 동일합니다.

</SwaggerDescription>

### Parameters

#### Query

<SwaggerParameter name="payment_request_date" type="String" required="true">
<mark style="color:red;">**요청일자**</mark>

YYYYMMDD 형식의 조회 날짜를 지정하시면 됩니다.

</SwaggerParameter>
<SwaggerParameter name="cid" type="String" required="false">
**카카오페이 MID**

포트원 내 설정된 카카오페이 CID를 기본값으로 합니다.

</SwaggerParameter>
<SwaggerParameter name="page" type="integer" required="false">
**페이지 번호**

</SwaggerParameter>
