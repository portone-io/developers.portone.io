## 베네피아 포인트를 이용하여 결제를 요청합니다.

<Swagger method="post" path="/benepia/payment" baseUrl="https://api.iamport.kr" summary="베네피아 포인트 결제요청">
  <SwaggerDescription>
    사용자로부터 전달받은 베네피아 계정 아이디, 비밀번호와 함께 결제정보를 요청하여 베네피아 포인트 사용처리합니다.

    **KCP를 통해서만 진행**

    되므로 KCP사이트코드 발급이 필요합니다.
  </SwaggerDescription>

  ### Parameters

  #### Body

  <SwaggerParameter name="benepia_user" type="String" required="true">
    <mark style="color:red;">
      **베네피아 계정 아이디**
    </mark>
  </SwaggerParameter>

  <SwaggerParameter name="benepia_password" type="String" required="true">
    <mark style="color:red;">
      **베네피아 계정 비밀번호**
    </mark>
  </SwaggerParameter>

  <SwaggerParameter name="merchant_uid" type="String" required="true">
    <mark style="color:red;">
      **고객사 주문번호**
    </mark>

    이미 결제가 이뤄진 적이 있는 merchant\_uid로는 결제요청이 불가능합니다.
  </SwaggerParameter>

  <SwaggerParameter name="amount" type="double" required="true">
    <mark style="color:red;">
      **결제요청금액**
    </mark>
  </SwaggerParameter>

  <SwaggerParameter name="name" type="String(40)" required="true">
    <mark style="color:red;">
      **주문명**
    </mark>
  </SwaggerParameter>

  <SwaggerParameter name="buyer_name" type="String(16)">
    **주문자명**
  </SwaggerParameter>

  <SwaggerParameter name="buyer_email" type="String(64)">
    **주문자 Email 주소**
  </SwaggerParameter>

  <SwaggerParameter name="buyer_tel" type="String(16)">
    **주문자 전화번호**
  </SwaggerParameter>

  <SwaggerParameter name="buyer_addr" type="String(128)">
    **주문자 주소**
  </SwaggerParameter>

  <SwaggerParameter name="buyer_postcode" type="String(8)">
    **주문자 우편번호**
  </SwaggerParameter>

  <SwaggerParameter name="pg" type="String">
    **PG구분코드**

    KCP PG설정이 2개 이상인 경우, 결제가 진행되길 원하는 KCP사이트코드를 지정하실 수 있습니다.

    pg 파라메터는 `kcp.{KCP사이트코드}` 형식의 문자열입니다.
    pg 파라메터가 누락되는 경우 PG설정 내 KCP설정을 자동 탐색하여 처리하게 됩니다.
  </SwaggerParameter>

  <SwaggerParameter name="notice_url" type="String">
    **Notification URL(Webhook URL)**

    선언되지 않으면 포트원 관리자 페이지에 정의된 Notification URL값을 사용합니다
  </SwaggerParameter>

  <SwaggerParameter name="custom_data" type="Array">
    **결제정보와 함께 저장할 추가정보**

    객체로 전달되는 경우 JSON 문자열로 저장
  </SwaggerParameter>
