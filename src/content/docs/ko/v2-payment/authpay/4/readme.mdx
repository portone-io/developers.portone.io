title: 4. 결제완료 처리하기 (클라이언트)
description: 결제결과를 검증하여 완료 처리를 할 수 있는 가이드입니다.
---

결제창이 활성화되는 방식에 따라 결제결과를 획득하는 방법이 상이합니다. 일반적으로 PC 환경에서는 iframe 혹은 popup 방식으로 결제창이 활성화되며 모바일 환경에서는 새로운 페이지로 리다이렉트 되어 결제창이 활성화 됩니다.

결제가 정상적으로 완료되면 결제창 형태에 따라 아래와 같이 결제결과를 받아 볼 수 있습니다.

| iframe / popup 방식 |  redirect 방식  |
| :---------------: | :-------------: |
| **callback 함수** | **redirectUrl** |

자세한 가이드는 아래 링크를 참고해주세요.

결제완료 처리를 위해서는 [3](../3 "mention") 을 완료하셔야 합니다.
