---
title: 3. 인증 완료 정보 전달하기
description: 본인인증 이후 획득된 정보를 처리하는 방법을 안내합니다.
---

본인인증 창이 활성화되는 방식에 따라 본인인증 결과 획득하는 방법이 상이합니다. 일반적으로 PC 환경에서는 popup 방식으로 본인인증 창이 활성화되며 모바일 환경에서는 새로운 페이지로 리다이렉트 되어 본인인증 창이 활성화 됩니다.

결제가 정상적으로 완료되면 본인인증창 형태에 따라 아래와 같이 결제결과를 받아 볼 수 있습니다.

| popup 방식 |  redirect 방식  |
| :---------------: | :-------------: |
| **callback 함수** | **redirectUrl** |

자세한 가이드는 아래 링크를 참고해주세요.

- [popup 방식에서의 결과 처리](./popup)
- [redirect 방식에서의 결과 처리](./redirect)
