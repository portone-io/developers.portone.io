하위 페이지에서 각 PG사별 결제 연동 방법을 확인할 수 있습니다.

## PG사별 지원되는 결제 수단 정보

### 결제

- **토스페이먼츠**

  - 연동 기능: 일반인증결제(결제창) / 비인증결제(API) / 정기결제(결제창/API) / 간편결제(결제창/다이렉트 호출)

  - 결제 수단

    - 일반인증결제: 카드 / 실시간 계좌이체 / 가상계좌 / 휴대폰 결제
    - 비인증결제(API): 카드 / 가상계좌
    - 정기결제(결제창/API): 카드
    - 간편결제(카드 결제창 노출방식): 카카오페이 / 네이버페이 / 토스페이 / 삼성페이 / SSGpay / L.Pay / LGpay
    - 간편결제(다이렉트 호출방식): 카카오페이 / 네이버페이 / 토스페이 / 삼성페이 / SSGpay / L.Pay / LGpay

- **KSNET**

  - 연동 기능: 일반인증결제(결제창) / 비인증결제(API) / 정기결제(API) / 간편결제(결제창/다이렉트 호출)

  - 결제 수단

    - 일반인증결제: 카드 / 실시간 계좌이체 / 가상계좌 / 휴대폰 결제
    - 비인증결제(API): 카드 / 가상계좌
    - 정기결제(API): 카드
    - 간편결제(카드 결제창 노출방식): 카카오페이 / 네이퍼페이 / 페이코 / SSG Pay / L.Pay
    - 간편결제(결제창내 간편결제 지정 노출방식): 카카오페이 / 네이퍼페이 / 페이코 / SSG Pay / L.Pay

- **카카오페이**

  - 연동 기능: 간편결제(결제창)

  - 결제 수단

    - 간편결제(카드 결제창 노출방식): 카카오페이

- **네이버페이**

  - 연동 기능: 간편결제(결제창)

  - 결제 수단

    - 간편결제(카드 결제창 노출방식): 네이버페이

- **스마트로**

  - 연동 기능: 일반인증결제(결제창) / 비인증결제(API) / 정기결제(API) / 간편결제(결제창/다이렉트 호출)

  - 결제 수단

    - 일반인증결제: 카드 / 실시간 계좌이체 / 가상계좌 / 휴대폰 결제
    - 비인증결제(API): 가상계좌
    - 정기결제(결제창/API): 카드
    - 간편결제(카드 결제창 노출방식): 카카오페이 / 네이퍼페이 / 페이코 / Pin Pay / L.Pay
    - 간편결제(다이렉트 호출방식): 카카오페이 / 네이퍼페이 / 페이코 / Pin Pay / L.Pay

- **나이스페이먼츠**

  - 연동 기능: 일반인증결제(결제창) / 비인증결제(API) / 정기결제(API) / 간편결제(결제창/다이렉트 호출)

  - 결제 수단

    - 일반인증결제: 카드 / 실시간 계좌이체 / 가상계좌 / 상품권 / 휴대폰 결제
    - 비인증결제(API): 카드 / 가상계좌
    - 정기결제(API): 카드
    - 간편결제(카드 결제창 노출방식): 카카오페이 / 네이퍼페이 / 페이코 / SSG Pay / 11Pay (구.SKPay) / L.Pay / 삼성페이 / 애플페이
    - 간편결제(다이렉트 호출방식): 카카오페이 / 네이퍼페이 / 페이코 / SSG Pay / 11Pay (구.SKPay) / L.Pay / 삼성페이 / 애플페이

- **페이팔**

  - 연동 기능: 일반인증결제(결제창) / 정기결제(결제창)

  - 결제 수단

    - 일반인증결제 : 페이팔 (페이팔 내에서 카드, 계좌 등 결제 수단 선택 가능)
    - 정기결제(결제창) : 페이팔 (페이팔 내에서 카드, 계좌 등 결제 수단 선택 가능)

### 본인인증

- **다날 본인인증**

  - 연동 기능: SMS 문자 메세지를 통한 본인인증 / PASS APP을 통한 본인인증

  - 본인인증 수단

    - SDK 본인인증: SMS / PASS
    - API 본인인증: SMS / PASS
