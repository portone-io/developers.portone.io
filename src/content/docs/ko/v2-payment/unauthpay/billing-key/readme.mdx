---

import Hint from "~/components/gitbook/Hint.astro";

<Hint style="info">

**빌링키란?**

구독형 정기결제, 종량제 과금결제 등 원하는 시점에 재 결제를 진행할 수 있는 결제용 암호화 키 입니다.

고객사가 고객의 카드정보를 소유할 수 없기 때문에 카드사로부터 해당 카드에 대응하는 빌링키 발급 받아 저장하고 원하는 시점에 해당 빌링키로 결제를 청구할 수 있습니다.

</Hint>

<Hint style="info">

**빌링키 결제란?**

결제수단 정보로 빌링키 발급 후 빌링키를 이용하여 단건/예약 결제가 이루어지는 방식입니다.

정기 구독 서비스와 같이 구매자가 결제 수단 정보를 최초 입력 후 정기적으로 자동 결제를 일으키고 싶은 경우 해당 방식을 이용합니다.

</Hint>
