## 비인증 결제란?

실시간 인증 없이 결제수단의 결제 정보만으로 결제를 요청하는 방식을 통칭합니다.

<Hint style="info">
구매자의 인증 없이도 결제를 진행할 수 있어 간편하다는 장점이 있지만 구매자 인증 과정이 생략되는만큼 서비스 운영 중 보안상의 이슈가 발생할 확률이 높아 **PG 가입 시 입점심사가 까다롭게 진행**된다는 점은 참고해주세요.

</Hint>

### 1. 빌링키 결제

**결제수단 정보로 빌링키 발급 후 빌링키를 이용하여 단건/예약 결제가 이루어지는 방식**

정기 구독 서비스와 같이 구매자가 결제 수단 정보를 최초 입력 후 정기적으로 자동 결제를 일으키고 싶은 경우 해당 방식을 이용합니다.

### 2. 키인(수기) 결제

**수기(키인) 결제 : 구매자 인증 없이 결제수단 정보만으로 결제가 이루어지는 방식**

수기(키인) 결제는 구매자의 별도 인증절차 없이 결제수단 정보만으로 결제가 이루어지는 방식입니다. 구매자로부터 입력받은 결제수단 정보로 일회성 결제를 일으키는 경우 해당 방식을 이용합니다.
