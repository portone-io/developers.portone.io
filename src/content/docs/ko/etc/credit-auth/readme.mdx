  신용카드본인인증 서비스는 현재 서비스 중단으로 신규계약이 불가합니다.\
  (기존에 이용중인 고객사는 지속 사용 가능)
</Hint>

**신용카드 본인인증 서비스란** 신용카드 인증을 통하여 해당 신용카드 소지자의 개인정보(이름, 생년월일, 성별, 외국인여부, KISA 고유식별값)를 확인할 수 있는 서비스입니다.
