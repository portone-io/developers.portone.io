### 통합인증을 연동할 페이지에 포트원 라이브러리를 추가합니다.

<Hint style="info">
신용카드 본인인증 기능은 **포트원 JavaScript v1.1.8**부터 지원합니다.

</Hint>

<Tabs>
<Tab title="HTML">
```html title="client-side"
<!-- jQuery -->
<script type="text/javascript" src="https://code.jquery.com/jquery-1.12.4.min.js" ></script>
<!-- iamport.payment.js -->
<script type="text/javascript" src="https://cdn.iamport.kr/js/iamport.payment-{SDK-최신버전}.js"></script>
```

</Tab>
</Tabs>

<Hint style="danger">
**jQuery 1.0 이상이 설치**되어 있어야 합니다.

</Hint>

### 본인인증 페이지에 [<mark style="color:blue;">`가맹점 식별코드`</mark>](../../ready/3.)를 이용하여 `IMP` 객체를 초기화합니다.

<Tabs>
<Tab title="JavaScript">
```javascript title="client-side"
var IMP = window.IMP; // 생략 가능
IMP.init("{가맹점 식별코드}"); // 예: imp00000000
```

</Tab>

<Tab title="JavaScript(ES Next)">
```javascript title="Client-side"
var IMP = window.IMP; // 생략 가능

```

</Tab>
