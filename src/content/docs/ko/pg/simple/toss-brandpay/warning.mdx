# 현금영수증 유의사항

### 아래의 경우에는, 현금영수증이 자진발급, 소득공제로 발급됩니다.

- 현금영수증을 결제창에서 미발행 하는경우
- 브랜드페이의 `methodId` 를 전달하여 결제하며, `bypass.cashReceiptType`을 `anonymous`로 전달하는 경우

# 카드 결제시 유의사항

### 할부 개월수 리스트를 다른 용도로 사용함

다른 PG사는 `display: { card_quota }` 파라미터로 구매자에게 노출 될 할부 개월수 리스트 제어가 가능하나 브랜드페이에서는 해당 파라미터를 할부개월 지정으로 사용합니다.
아래의 경우, 6개월로 할부결제가 진행됩니다.

- 다른PG사에서는 6개월만 선택할 수 있도록 허용되는 값이지만, 브랜드페이에서는 6개월 할부로 지정하는 값입니다.

```jsx
display: {
  card_quota: [6],
},
```

# 취소관련 유의사항

### 잔여 면세금액 계산

부분취소 및 전체취소시, 남은 면세금액을 정확하게 계산하여 요청해야합니다.

# 테스트 관련 유의사항

### 토스의 테스트키 이용시 더미값이 내려옴

브랜드페이의 경우, 토스페이먼츠의 정책상, 테스트 환경의 API 키를 이용하는경우, 더미값이 내려옵니다.
연동이 정상적으로 완료되었는지는 라이브 API키 를 이용해서 확인해야 합니다.

### 토스와 API를 공유함

취소를 제외한 현금영수증 발급, 결제조회 API는 기존에 존재하는 토스페이먼츠의 API를 이용해야합니다.
