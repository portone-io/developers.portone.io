KCP 사용을 위해서는 KCP와 PG-API 계약이 필요하며, 해당 계약 이후에 하단의 절차 진행이 가능합니다.

  1. [KCP 가맹점관리자](https://partner.kcp.co.kr/) 접속 후 로그인을 합니다.

  2. \[고객센터]→\[인증센터]→\[KCP PG-API]를 클릭합니다.

  3. KCP PG-API 발급 및 조회 하단의 서비스 인증서 발급을 클릭합니다.

     - 개인 키 비밀번호 입력 이후에 PG-API 인증서, PG-API 개인키 파일을 다운로드 받을 수 있습니다.
     - 비밀번호 분실 시에는 폐기 후 재발급이 필요하며, 인증서, 개인키만 분실 했을 시에는 관리자 페이지 비밀번호를 통해 재다운로드 가능합니다.
     - 이 과정에서 발급한 PG-API 인증서, PG-API 개인키, 개인키 비밀번호를 크레덴셜로 확인합니다.
<Figure src={image1} caption="KG이니시스 가맹점관리자 내 크레덴셜 정보 조회 화면 1" />

  4. (정기결제 계약사 한정) 빌링키 발급을 위해 필요한 정기결제 그룹 아이디를 발급합니다.
     - \[결제관리]→\[자동결제]→\[그룹 관리]→\[그룹 등록]을 진행합니다. 등록 이후 그룹ID를 크레덴셜로 사용합니다.
<Figure src={image2} caption="KG이니시스 가맹점관리자 내 크레덴셜 정보 조회 화면 1" />

  5. 포트원 콘솔에서 채널 추가 시 \[상점아이디]와 위 과정에서 발급받은 크레덴셜 값들을 입력한 후 `저장`을 클릭합니다.
</VersionGate>
