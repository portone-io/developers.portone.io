1. [KG이니시스 가맹점관리자](https://iniweb.inicis.com/security/login.do) 접속 후 로그인을 합니다.

2. \[상점정보]→\[계약정보]→\[부가정보]를 클릭합니다.

3. 웹결제 signkey 생성 조회의 조회 버튼을 클릭하여 값을 확인할 수 있습니다. 값이 표시되지 않는 경우 생성/갱신 버튼을 클릭해 주세요.

4. INIAPI key 생성 갱신의 조회 버튼을 클릭하여 \[INIAPI KEY], \[INIAPI IV] 값을 확인할 수 있습니다.
   값이 표시되지 않는 경우 갱신 버튼을 클릭해 주세요.

<Figure src={image15} caption="KG이니시스 가맹점관리자 내 크레덴셜 정보 조회 화면 1" />

5. INILite Key의 조회 버튼을 클릭하여 값을 확인할 수 있습니다.

<Figure src={image23} caption="KG이니시스 가맹점관리자 내 크레덴셜 정보 조회 화면 2" />

6. 포트원 콘솔에서 채널 추가 시 \[상점아이디], \[웹결제 signkey]를 입력한 후 `저장`을 클릭합니다.

\[INILite Key], \[INIAPI Key], \[INIAPI IV]의 값은 케이스에 따라 추가로 입력한 후 `저장`을 클릭합니다.
