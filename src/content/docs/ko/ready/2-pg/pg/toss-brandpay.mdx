<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. [결제 연동] 메뉴 클릭
3. [결제대행사 설정 및 추가]에서 [테스트] → [토스페이먼츠] → [토스페이먼츠 브랜드페이] 선택 후 오른쪽 **추가** 버튼 클릭

import image1 from "./_assets/toss-brandpay/brandpay-channel-1.png";

<Figure src={image1} caption="테스트 환경 설정 예시" />

4. 토스페이먼츠에서 발급받은 테스트상점 정보를 입력 후 저장합니다.

- **채널 이름**은 채널을 구분하기 위한 용도로 한글/영문 관계없이 입력 가능합니다.
- **과세 구분(부가세 신고자료용)** 설정은 정산 서비스 이용 시 사용되는 설정입니다.

import image2 from "./_assets/toss-brandpay/brandpay-channel-2.png";

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

5. 토스페이먼츠 개발자센터에 리다이렉트 URL 설정

- 토스페이먼츠 개발자센터의 [브랜드페이 페이지](https://developers.tosspayments.com/my/brandpay)에서 테스트용 리다이렉트 URL을 추가해주세요.
- 리다이렉트 URL은 반드시 `https://tx-gateway-service.prod.iamport.co/toss-brandpay/callback-auth?channelKey={포트원 결제 연동의 채널키}`로 설정해주셔야 합니다.

import image3 from "./_assets/toss-brandpay/brandpay-redirect-url-test.png";

<Figure src={image3} caption="테스트 환경 리다이렉트 URL 설정" />

#### 실 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. [결제 연동] 메뉴 클릭
3. [결제 대행사 설정 및 추가]에서 [실 연동] → [토스페이먼츠] → [토스페이먼츠 브랜드페이] 선택 후 오른쪽 **추가** 버튼 클릭

import image4 from "./_assets/toss-brandpay/brandpay-channel-3.png";

<Figure src={image4} caption="실 환경 설정 예시" />

4. 카드사 심사 완료 후 토스페이먼츠에서 발급받은 실 상점 정보를 입력 후 저장합니다.

- **채널 이름**은 채널을 구분하기 위한 용도로 한글/영문 관계없이 입력 가능합니다.
- **과세 구분(부가세 신고자료용)** 설정은 정산 서비스 이용 시 사용되는 설정입니다.

import image5 from "./_assets/toss-brandpay/brandpay-channel-4.png";

<Figure src={image5} caption="실 환경 상세 설정 예시" />

5. 토스페이먼츠 개발자센터에 리다이렉트 URL 설정

- 토스페이먼츠 개발자센터의 [브랜드페이 페이지](https://developers.tosspayments.com/my/brandpay)에서 라이브용 리다이렉트 URL을 추가해주세요.
- 리다이렉트 URL은 반드시 `https://tx-gateway-service.prod.iamport.co/toss-brandpay/callback-auth?channelKey={포트원 결제 연동의 채널키}`로 설정해주셔야 합니다.
