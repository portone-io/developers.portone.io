#### 포트원을 통한 알리페이 결제는 나이스페이먼츠 서비스를 이용합니다.

### 일반결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성방법
1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 알리페이 → 알리페이 선택 후 오른쪽 **추가** 버튼 클릭

import image1 from "./_assets/alipay/alipay-channel-guide-1.png";

<Figure src={image1} caption="테스트 설정 예시" />

4. 자동으로 입력된 값을 확인 후 저장합니다.

- 결제 채널 이름 속성은 원하시는 이름으로 한글/영문 관계없이 작성하시면 됩니다.
- 과세 구분(부가세 신고자료용) 설정은 정산 서비스 이용 시 설정이 필요합니다.

import image2 from "./_assets/alipay/alipay-channel-guide-2.png";

<Figure src={image2} caption="테스트 상세 설정 예시" />

</Tab>

<Tab title="실 결제">
#### **실** 환경 구성방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 실연동 → 알리페이 → 알리페이 선택후 오른쪽 **추가** 버튼 클릭

import image3 from "./_assets/alipay/alipay-channel-guide-3.png";

<Figure src={image3} caption="실 계정 설정 예시" />

4. 카드사 심사완료 후 나이스페이먼츠에서 발급받은 실상점 정보를 설정합니다.

- 결제 채널 이름 속성은 원하시는 이름으로 한글/영문 관계없이 작성하시면 됩니다.
- 과세 구분(부가세 신고자료용) 설정은 정산 서비스 이용 시 설정이 필요합니다.

import image4 from "./_assets/alipay/alipay-channel-guide-4.png";

<Figure src={image4} caption="실 계정 상세 설정 예시" />

</Tab>
