## 일반 결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 카카오페이 → 카카오페이 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image1} caption="테스트 환경 설정 예시" />

4. 카카오페이 결제창 테스트용 CID <mark style="color:red;">**TC0ONETIME**</mark> 를 확인하고 **저장**

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 카카오페이 → 카카오페이 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image3} caption="실 환경 설정 예시" />

4. 카카오페이 계약후 발급받은 <mark style="color:red;">**일반결제용 CID**</mark> 값을 기재 후 <mark style="color:red;">**저장**</mark>합니다.

<Figure src={image4} caption="실 환경 상세 설정 예시" />

</Tab>
