**네이버페이는 테스트 연동정보를 별도로 제공하지 않습니다. 가입신청 후 연동정보를 받아서 테스트 모드로 연동/개발 할 수 있습니다.**

</Hint>

## 일반 결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 네이버페이 주문형 → 네이버페이 주문형 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image1} caption="테스트 환경 설정 예시" />

4. 네이버페이에서 발급받은 연동정보를 입력 후 저장

- PG상점아이디(파트너ID) : 네이버페이 파트너ID를 입력합니다.
- 가맹점 인증키 : 네이버페이측에서 발급받은 가맹점 인증키를 입력합니다.

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

네이버페이 실 검수가 시작되면 채널의 유형을 **실 연동**으로 변경하고 동일하게 결제채널을 등록하시면 됩니다.

<Figure src={image3} caption="실연동 변경 예시" />

</Tab>
