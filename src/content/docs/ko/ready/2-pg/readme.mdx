포트원 회원가입 이후 관리자 페이지 내 **결제 연동** **->** **테스트 연동 관리** 탭에서 연동하고자 하는 PG사를 선택합니다.

**테스트 연동 관리** 메뉴에서는 포트원에서 지원하는 각 PG사별 테스트 설정을 진행할 수 있습니다.
