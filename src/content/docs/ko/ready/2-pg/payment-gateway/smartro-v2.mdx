**실 환경 구성 방법**

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결**제 연동** → **실 연동 관리** 탭
3. 결제 대행사 설정에서 (신) 스마트로 선택 후 오른쪽 `+ 추가` 버튼 클릭
   ![Untitled](https://s3-us-west-2.amazonaws.com/secure.notion-static.com/9b6d42ea-d39e-4991-b54f-92ba00663b98/Untitled.png)
4. 스마트로에서 전달받은 실 연동정보 입력 후 오른쪽 하단 `저장` 버튼 클릭

- **MID**: [스마트로 어드민페이지](https://www.smilepay.co.kr/) > 가맹점 > 가맹점정보 > 기본정보 에서 "MID"항목에서 확인가능
- **Merchant Key**: [스마트로 어드민페이지](https://www.smilepay.co.kr/) > 가맹점 > 가맹점정보 > Key관리 에서 확인가능
- **거래취소 비밀번호**: [스마트로 어드민페이지](https://www.smilepay.co.kr/) 가맹점 > 가맹점정보 > 취소비밀번호 설정한 값 그대로 세팅.
