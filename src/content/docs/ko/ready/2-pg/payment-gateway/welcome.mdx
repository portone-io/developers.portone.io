<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `웰컴페이먼츠` → `웰컴페이먼츠` 선택 후 `+ 추가` 버튼 클릭
4. 자동으로 설정된 MID 값 확인 후 저장

   - **웰컴페이먼츠 일반 결제 테스트용**
     - mid: `welcometst`
   - **웰컴페이먼츠 정기 결제 테스트용**
     - mid: `wpaybill01`

<Tab title="실 결제">

#### 실 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `실 연동` → `웰컴페이먼츠` → `웰컴페이먼츠` 선택 후 `+ 추가` 버튼 클릭
4. 카드사 심사 완료 후 웰컴페이먼츠에서 발급받은 실 상점 정보를 설정합니다.

   <Figure
     src={screenshot3}
     caption="실 연동 -> 웰컴페이먼츠 -> 웰컴페이먼츠 추가"
   />

5. 웰컴페이먼츠에서 전달받은 실 연동정보 입력 후 오른쪽 하단 `저장` 버튼 클릭

   <Figure src={screenshot4} width="375" caption="실 상점 정보 설정" />

- **MID**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 기본정보 "상점 ID" 항목에서 확인가능
- **Sign Key**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 부가정보 > 웹결제 signkey 생성조회 "조회" 버튼 클릭하여 확인가능
- **필드 암호화 IV**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 부가정보 > 웹결제 signkey 생성조회 "조회" 버튼 클릭하여 확인가능
- **필드 암호화 Key**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 부가정보 > 웹결제 signkey 생성조회 "조회" 버튼 클릭하여 확인가능
