<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. **결제 연동** → **테스트 연동 관리** 탭
3. 결제 대행사 설정에서 웰컴페이먼츠 선택 후 오른쪽 `+ 추가` 버튼 클릭

   <Figure src={screenshot1} />

4. 테스트 연동 정보 확인 후 오른쪽 하단 `저장` 버튼 클릭

- **웰컴페이먼츠 일반 결제 테스트용**
  - mid: `welcometst`
- **웰컴페이먼츠 정기 결제 테스트용**
  - mid: `wpaybill01`

5. 테스트 연동 정보 확인 후 오른쪽 하단 `저장` 버튼 클릭

<Tab title="실 결제">

#### 실 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. **결제 연동** → **실 연동 관리** 탭
3. 결제 대행사 설정에서 웰컴페이먼츠 선택 후 오른쪽 `+ 추가` 버튼 클릭

   <Figure src={screenshot3} />

4. 웰컴페이먼츠에서 전달받은 실 연동정보 입력 후 오른쪽 하단 `저장` 버튼 클릭

- **MID**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 기본정보 "상점 ID" 항목에서 확인가능
- **Sign Key**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 부가정보 > 웹결제 signkey 생성조회 "조회" 버튼 클릭하여 확인가능
- **필드 암호화 IV**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 부가정보 > 웹결제 signkey 생성조회 "조회" 버튼 클릭하여 확인가능
- **필드 암호화 Key**: [웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 부가정보 > 웹결제 signkey 생성조회 "조회" 버튼 클릭하여 확인가능

<Figure src={screenshot4} width="375" />

<Hint style="info">
**가상계좌 사용시 주의사항**

[웰컴페이먼츠 상점관리자페이지](https://wbiz.paywelcome.co.kr/) > 상점정보 > 결제수단 정보 > 가상계좌 에서 **입금통보 URL** 설정을 반드시 진행해 주셔야 합니다. (누락시 입금 완료된 내역이 반영되지 않습니다.)
자세한 내용은 [가상계좌 입금통보설정](https://developers.portone.io/docs/ko/result/notice) 에서 확인할 수 있습니다.
