## SMS 본인인증

<Tabs>
<Tab title="테스트 환경">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `다날` → `다날 휴대폰결제/본인인증` 선택
4. 채널속성에서 `본인인증` 선택 후 `+ 추가` 버튼 클릭
5. 카드사 심사 완료 후 다날에서 발급받은 실 상점 정보 기재 후 저장

<Hint style="warning">
**다날 SMS 본인인증**

휴대폰 SMS 본인인증은 테스트는 별도의 테스트 전용 CPID 정보를 제공하지 않습니다.

테스트 진행을 위해서는 다날 계약 후 발급받은 정보로만 가능합니다.

</Hint>

<Figure src={image9} />

</Tab>

#### 실 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `실 연동` → `다날` → `다날 휴대폰결제/본인인증` 선택
4. 채널속성 본인인증 선택 후 `+ 추가` 버튼 클릭
5. 카드사 심사 완료 후 다날에서 발급받은 실상점 정보 기재 후 저장

<Hint style="info">
**다날 연동 시 주의사항**

- 다날의 경우 하나의 상점아이디(CPID)에 인증/비 인증 결제 설정이 가능합니다.
- 다날 정책상 계좌이체/가상계좌는 테스트 상점아이디가 제공되지 않습니다. 테스트를 위해서는 실제 계약 후 발급받은 상점아이디로 환경 구성 후 테스트 결제를 해야합니다.
- SMS 본인인증은 통신사 정책으로 인하여 테스트 모드 이용이 불가합니다. 테스트를 위해서는 실제 계약 후 발급받은 상점아이디로 환경 구성 후 테스트를 진행해야 합니다.
- 테스트 환경에서 부분취소는 불가능합니다.

</Hint>
