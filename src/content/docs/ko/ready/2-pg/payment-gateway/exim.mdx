<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 엑심베이 → 엑심베이 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image1} caption="테스트 환경 설정예시" />

4. 자동으로 입력된 값 확인 후 저장

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

> - 엑심베이 테스트 모드의 경우 결제 시 실제 출금이 이루어지지 않습니다.
> - 테스트모드에서는 테스트용 카드를 사용해야 하며 해외에서 발급받은 실제 카드를 사용하는 경우 오류가 발생됩니다.
> - [엑심베이에서 제공하는 해외결제용 테스트 카드 확인하기](https://support.eximbay.com/hc/ko/articles/360059014854-%EA%B2%B0%EC%A0%9C%ED%85%8C%EC%8A%A4%ED%8A%B8%EB%A5%BC-%EC%A7%84%ED%96%89%ED%95%B4%EB%B3%B4%EA%B3%A0-%EC%8B%B6%EC%9D%80%EB%8D%B0-%ED%95%B4%EC%99%B8%EC%B9%B4%EB%93%9C%EA%B0%80-%EC%97%86%EC%8A%B5%EB%8B%88%EB%8B%A4-%EC%96%B4%EB%96%BB%EA%B2%8C-%ED%95%A0-%EC%88%98-%EC%9E%88%EB%82%98%EC%9A%94-)

</Tab>

<Tab title="실 결제">
#### **실** 환경 구성방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 실연동 →엑심베이 → 엑심베이 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image3} caption="실 환경 설정 예시" />

4. 카드사 심사완료 후 엑심베이에서 발급받은 실상점 정보 입력 후 저장

<Figure src={image4} caption="실 환경 상세 설정 예시" />

</Tab>
</Tabs>

#### **참고사항**

포트원은 엑심베이 비인증 결제 방식을 지원하지 않습니다.

</Hint>
