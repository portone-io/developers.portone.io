<Hint style="warning">
**나이스페이먼츠 구 모듈 연동 방법에 관한 매뉴얼**입니다.

신규 연동 가맹점이시거나 신 모듈로의 업그레이드를 원하신다면 [(신) 나이스페이먼츠 연동 문서](nice-v2)로 이동해주세요.

</Hint>

## 인증 결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `나이스페이먼츠` → `(구) 나이스페이먼츠` 선택 후 `+ 추가` 버튼 클릭
4. 자동으로 설정된 MID 값 확인 후 저장

> 테스트 연동 설정시 포트원 테스트 MID 계정으로 결제가 발생됩니다.

<Figure
  src={image1}
  caption="테스트 -> 나이스페이먼츠 -> (구) 나이스페이먼츠 추가"
/>

<Figure src={image2} width="375" />

나이스페이먼츠 테스트 모드에서는 아래 결제 방식을 사용할 수 없습니다.

- 신용카드 : **NH농협카드** , **KB국민카드** 이용불가
- 휴대폰 소액결제 : **KT 통신사** 이용불가

<Hint style="info">
**나이스페이먼츠 테스트 모드의 경우 매일 23:00\~23:50분 사이 자동 취소됩니다.**

취소된 내역은 포트원에 자동 반영되지 않으므로 되도록 테스트 결제 후 결제 취소 요청 진행을 권장합니다.

</Hint>

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `실 연동` → `나이스페이먼츠` → `(구) 나이스페이먼츠` 선택 후 `+ 추가` 버튼 클릭
4. 카드사 심사 완료 후 나이스페이먼츠에서 발급받은 실 상점 정보를 설정합니다.

/>

<Hint style="info">
**가상계좌 사용시 주의사항**

[나이스페이 상점관리자페이지](https://npg.nicepay.co.kr/logIn.do) > 가맹점정보 > 기본정보 > 결제데이터통보 에서 **가상계좌 입금통보설정 URL** 설정을 반드시 진행해 주셔야 합니다. (누락시 입금 완료된 내역이 반영되지 않습니다.)
자세한 내용은 [가상계좌 입금통보설정](https://developers.portone.io/docs/ko/result/notice) 에서 확인할 수 있습니다.

</Hint>
