<Hint style="info">
**나이스페이먼츠-신모듈 연동 안내**

포트원 나이스페이먼츠 연동은 신모듈/구모듈 연동 방식을 제공하고 있습니다.
신모듈에서 더 많은 기능을 제공하고 있으며, **원활한 기술 지원 및 더 나은 연동 경험**을 위해 **나이스페이먼츠-신모듈로 연동**하시는 것을 권장드립니다. <br/>
**포트원 V2 모듈에서는 나이스페이먼츠-신모듈만 지원합니다.**

</Hint>

## 인증 및 비인증결제 (나이스페이먼츠-신모듈)

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `나이스페이먼츠` → `(신)나이스페이먼츠` 선택 후 `+ 추가` 버튼 클릭
4. 자동으로 설정된 MID 값 확인 후 저장

    - **(신) 나이스페이먼츠 인증결제 가상계좌 테스트**
      - mid : `nictest00m`
    - **(신) 나이스페이먼츠 인증결제 테스트**
      - mid : `iamport00m`
    - **(신) 나이스페이먼츠 비인증결제 테스트**
      - mid : `iamport01m`

<Figure
  src={image6}
  caption="테스트/실 연동 관리 → 테스트 선택 → 나이스페이먼츠 선택 → (신) 나이스페이먼츠 → 추가"
/>


<Figure src={image7} caption="자동으로 세팅된 MID 및 연동 정보를 확인 후 저장 클릭" />

<Hint style="info">
**나이스페이먼츠 테스트 모드에서는 아래의 결제수단은 테스트가 불가능합니다.**

- 신용카드 : NH농협카드 , KB국민카드 결제 불가
- 휴대폰 소액결제 : KT 통신사 결제 불가
</Hint>

<Hint style="info">
**나이스페이먼츠 테스트 모드의 경우 매일 23:00\~23:50분 사이 자동 취소됩니다.**

취소된 내역은 포트원으로 자동 반영되지 않기 때문에 되도록 테스트 이후 포트원 콘솔내에서 결제 취소 진행을 권장드립니다. 

</Hint>

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `실 연동` → `나이스페이먼츠` → `(신)나이스페이먼츠` 선택 후 `+ 추가` 버튼 클릭
4. 카드사 심사 완료 후 나이스페이먼츠에서 발급받은 실 상점 정보를 설정합니다.

<Figure src={image8} caption="테스트/실 연동 관리 → 실 연동 선택 → 나이스페이먼츠 선택 → (신) 나이스페이먼츠 → 추가" />

<Figure
  src={image9}
  caption="카드사 심사 완료 후 나이스페이먼츠에서 발급받은 실 상점 정보 설정"
/>

<Hint style="info">
**가상계좌 사용시 유의사항**

나이스페이먼츠 관리자 페이지에 접속하여 **가맹점정보 > 기본정보 > 결제데이터통보 메뉴에서 가상계좌 입금통보 URL**을 반드시 설정해 주셔야 합니다. (누락시 입금된 내역이 반영되지 않습니다.)

</Hint>

<Hint style="info">
**거래취소비밀번호 확인방법**

- 거래취소 비밀번호 :  [나이스페이 상점관리자페이지](https://npg.nicepay.co.kr/logIn.do) > 가맹점정보 > 비밀번호관리 > 거래취소비밀번호 에서 설정(최대 10자리)해 주셔야 합니다.
</Hint>


</Tab>
