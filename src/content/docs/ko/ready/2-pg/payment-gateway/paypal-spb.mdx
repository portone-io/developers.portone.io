#### 구매자 계정 설정/조회 방법

- 페이팔 테스트용 구매자 계정 정보를 생성한 후 결제해야 합니다. [[→ 테스트용 계정생성 바로가기](https://developer.paypal.com/dashboard/accounts)]
- Sandbox Accounts의 Country가 `US`인 Personal(구매자) 계정으로 테스트해야 합니다.\
  판매자와 구매자 계정 국가가 `한국`인 경우 페이팔 정책상 결제가 불가하며, 판매자는 한국이고 구매자가 미국인 경우는 가능합니다.
- 페이팔 테스트 결제용 카드 정보를 확인한 후 테스트해야 합니다. \[[→ 테스트용 카드정보 바로가기](https://developer.paypal.com/api/rest/sandbox/card-testing/#link-creditcardgeneratorfortesting)]

#### 포트원 콘솔 내 채널 설정 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `페이팔` → `(신)페이팔` 선택 후 `+ 추가` 버튼 클릭
4. 원하는 판매자 국가 선택 후 자동으로 설정된 크레덴셜 확인 후 저장

<Hint style="info">
PG사를 여러개 이용중이신 경우, <br />
포트원 V1 결제모듈 이용 시 `pg` 파라미터에 `{ PG 구분코드 }.{ PG 상점 아이디 }` 형식으로 지정하여 호출되도록 구현해야 합니다. (ex. `paypal_v2.7WBB3CKT63FRG`) (참고 문서 \[[바로가기](../../../console/pg)]) <br />
포트원 V2 결제모듈 이용 시 `channelKey`를 지정하여 호출되도록 구현해야 합니다.

#### 포트원 콘솔 내 채널 설정 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `실 연동` → `페이팔` → `(신)페이팔` 선택 후 `+ 추가` 버튼 클릭
4. 페이팔 판매자 ID (Account ID)를 입력한 후 저장

## 판매자 계정 설정/조회 방법

PayPal 판매자 ID 조회 경로: [페이팔 홈페이지](https://www.paypal.com/businessmanage/account/aboutBusiness) 로그인 > 계정 설정 > 비즈니스 정보 > PayPal 판매자 ID 확인이 가능합니다.
