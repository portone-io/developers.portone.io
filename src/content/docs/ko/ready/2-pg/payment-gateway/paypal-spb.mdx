<Hint style="info">
페이팔은 별도의 계약없이 [포트원](https://admin.portone.io/)을 통해 가입신청 후 바로 서비스 이용이 가능합니다.
 
**(신) 페이팔의 경우 일반결제의 경우 SPB(Smart Payment Button) 방식으로 지원하며, 정기결제의 경우 RT(Reference Transaction) 방식으로 지원합니다.**
</Hint>


1. [https://admin.portone.io/](https://admin.portone.io/) 이메일 계정으로 회원가입
2. 로그인 후 **전자결제 신청** → **해외결제** 선택 → **페이팔** 선택 → 사업자정보 입력 및 사업자등록증 사본 업로드 → **신청하기**
3. 페이팔 홈페이지를 통한 추가 가입 진행 필수 (접수 처리시 신청 경로가 자동메일로 안내 됩니다. 해당 절차를 진행하셔야만 실운영모드 이용이 가능합니다.)

## 판매자 계정 설정/조회 방법

<Hint style="info">
포트원에서 제공하는 페이팔 테스트용 공용계정(Sandbox Accounts)으로 이용 해 주셔야 합니다.

</Hint>

페이팔 테스트용 판매자 ID: 아래 이용하실 국가별 “판매자ID”를 직접 세팅 해 주세요

테스트가 필요한 국가는 아래 `테스트용 판매자 ID 목록`을 참고 해 주시고, 추가로 테스트가 필요한 국가는 포트원 고객센터를 통해 별도 문의 주시기 바랍니다. (고객센터 상담톡 \[[바로가기](https://x06k7.channel.io/lounge)])

## 구매자 계정 설정/조회 방법

- 페이팔 테스트 결제용 구매자 계정 정보 생성하기 \[[바로가기](https://developer.paypal.com/dashboard/accounts)]
- Sandbox Accounts의 Country가 "US"인 Personal(구매자) 계정으로 테스트해야 합니다.\
  판매자와 구매자 계정 국가가 "한국"인 경우 페이팔 정책상 결제가 불가하며, 판매자는 한국이고 구매자가 미국인 경우는 가능합니다.
- 페이팔 테스트 결제용 카드 정보 \[[바로가기](https://developer.paypal.com/api/rest/sandbox/card-testing/#link-creditcardgeneratorfortesting)]

## 콘솔 설정 방법

- [포트원 관리자 콘솔](https://admin.portone.io/) → 결제 연동→ 테스트/실 연동 관리 → 테스트 / 페이팔 V2 / 페이팔 V2 선택 → \[추가] 버튼 클릭 → ‘PayPal 판매자 ID’ 세팅 → \[저장] 클릭
