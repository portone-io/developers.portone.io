## 인증**결제**

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 테스트 → `KICC` → `KICC` 선택 후 `+ 추가` 버튼 클릭
4. 자동으로 기재되는 상점정보 확인 후 저장

<Figure src={image1} />

<Figure src={image2} />

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 실연동 → `KICC` → `KICC` 선택 후 `+ 추가` 버튼 클릭
4. 카드사 심사 완료 후 KICC 에서 발급받은 실 상점 정보 기재 후 저장
