## 인증결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `키움페이` → `키움페이` 선택 후 `+ 추가` 버튼 클릭
4. 키움페이 인증 결제창 테스트 CTS17362 설정하기 클릭 및 MID 값 확인 후 저장

<Figure src={image2} />

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `실 연동` → `키움페이` → `키움페이` 선택 후 `+ 추가` 버튼 클릭
4. 키움페이 계약 이후 발급받은 상점 정보 기입 후 저장

<Figure src={image3} />

<Figure src={image4} />

<Hint style="success">
**확인사항**

- 상점아이디 발급 시 **판매 상품 유형**(실물 / 디지털)을 확인하시고 발급 요청해야 합니다.
  </Hint>

<Hint style="info">
**결제연동키 확인 방법**

- 결제연동키 : [키움페이 상점관리자페이지](https://agent.kiwoompay.co.kr/) > 고객지원 > 기술지원 > 연동정보설정 > CPID 선택 후 `조회하기` 클릭 > 결제 수단 선택
- 결제연동키는 결제 취소는 사용되므로 필수로 입력해야 합니다.

<Figure src={image7} />

</Hint>

<Hint style="info">
**키움페이 사용시 주의사항**

키움페이의 경우 각 결제수단별 **정상적인 결제 결과를 수신**하기 위해 키움페이 관리자 콘솔에서 반드시 아래 내용을 설정해야합니다.

1. [페이조아 관리자페이지](https://agent.kiwoompay.co.kr/) 로그인
2. 고객지원 > 기술지원 > 연동정보설정 페이지로 이동
   <Figure src={image8} />
3. 연동 정보 설정 화면에서 **CPID**를 선택 후 `조회하기` 버튼 선택
   <Figure src={image9} caption="연동정보 설정" />
4. 계약된 결제수단별로 하단의 **결과통지 URL** 입력 후 `수정하기` 버튼 클릭

> [https://service.iamport.kr/daou_payments/result](https://service.iamport.kr/daou_payments/result)

## 비인증 결제

<Tabs>
<Tab title="API 방식">
#### 정기결제 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `키움페이` → `키움페이` 선택 후 `+ 추가` 버튼 클릭
4. 키움페이 정기 API결제 테스트 CTS17266 설정하기 클릭 및 MID 값 확인 후 저장

<Figure src={image1} />

<Figure src={image5} />

#### 수기(키인)결제(1회성 결제) 테스트 환경 구성 방법

1. [포트원 관리자 콘솔](https://admin.portone.io/) 로그인
2. **결제 연동** 탭 클릭
3. `결제대행사 설정 및 추가` 폼에서 `테스트` → `키움페이` → `키움페이` 선택 후 `+ 추가` 버튼 클릭
4. 키움페이 수기(키인) API결제 테스트 CTS16628 또는 CTS16629 설정하기 클릭 및 MID 값 확인 후 저장

<Figure src={image1} />

<Figure src={image6} />

<Hint style="info">
**수기(키인) 테스트 환경 설정 유의사항**

키움페이의 경우 수기(키인) 결제 시 사용되는 필수 파라미터가 CPID 설정에 따라 달라지며, 포트원 테스트 환경에서는 아래 두 가지 설정을 제공합니다.

- **카드번호 + 비밀번호 + 유효기간 + 생년월일**
  - 위의 4. 번에서 키움페이 수기(키인) API결제 테스트 CTS16629 설정
- **카드번호 + 유효기간**
  - 위의 4. 번에서 키움페이 수기(키인) API결제 테스트 CTS16628 설정

</Hint>

<Hint style="success">
**확인사항**

- 포트원에서는 현재 키움페이의 경우 API 방식을 이용한 정기결제만 지원합니다.
- 키움페이 테스트 모드의 경우 실제 출금되지만 매일 23:00\~23:50분 사이 자동 취소됩니다.

</Hint>

4. 키움페이 계약 이후 상점 정보 기입 후 저장
