## 인증 및 비인증결제

<Tabs>
<Tab title="테스트 결제">
**테스트 환경 구성방법**

1. [포트원 관리자 콘솔](https://admin.portone.io/) → **결제 연동** → **테스트 연동 관리** → <mark style="color:red;">**나이스페이먼츠**</mark> 선택 → **(신) 나이스페이먼츠** → 추가

> 테스트 연동 설정시 포트원 테스트 MID 계정으로 결제가 발생됩니다.

> 포트원은 현재 나이스페이먼츠 비인증 결제의 경우 API 방식만 지원하고 있습니다.

<Figure
  src={image1}
  caption="결제 연동 → 테스트 연동 관리 → 나이스페이먼츠 선택 → (신) 나이스페이먼츠 → 추가"
/>
2. 자동으로 세팅된 MID 및 연동 정보를 확인한 후 **저장** 클릭

- (신) 나이스페이먼츠 일반결제 테스트 nictest00m

- (신) 나이스페이먼츠 비인증결제 테스트 nictest04m

<Figure src={image2} caption="자동으로 세팅된 MID 및 연동 정보를 확인 후 저장 클릭" />

(신) 나이스페이먼츠 테스트 모드에서는 다음과 같은 결제수단은 테스트가 불가능합니다. 

- 신용카드 : NH농협카드 , KB국민카드 결제 불가
- 휴대폰 소액결제 : KT 통신사 결제 불가

<Hint style="info">
**나이스페이먼츠 테스트 모드의 경우 매일 23:00\~23:50분 사이 자동 취소됩니다.**

취소된 내역은 포트원로 자동 반영되지 않기 때문에 되도록 테스트 이후 포트원 콘솔내에서 결제 취소 진행을 권장드립니다. 

</Hint>

</Tab>

<Tab title="실 결제">
**실 환경 구성방법**

1. **결제 연동** → **실 연동 관리** 선택
2. <mark style="color:red;">**나이스페이먼츠**</mark> 선택 → **(신) 나이스페이먼츠** → 추가
3. 카드사 심사 완료 후 나이스페이먼츠에서 발급받은 실 상점 정보를 설정합니다.

<Figure src={image3} caption="실 연동 -> (신)나이스페이먼츠 추가" />

<Figure
  src={image4}
  caption="카드사 심사 완료 후 나이스페이먼츠에서 발급받은 실 상점 정보 설정"
/>

<Hint style="info">
**가상계좌 사용시 유의사항**

나이스페이먼츠 관리자 페이지에 접속하여 **가맹점정보 > 기본정보 > 결제데이터통보 메뉴에서 가상계좌 입금통보 URL**을 반드시 설정해 주셔야 합니다. (누락시 입금된 내역이 반영되지 않습니다.)

</Hint>

<Hint style="info">
**거래취소비밀번호 확인방법**

- 거래취소 비밀번호 :  [나이스페이 상점관리자페이지](https://pg.nicepay.co.kr) > 가맹점정보 > 비밀번호관리 > 거래취소비밀번호 에서 설정(최대 10자리) 가능
</Hint>
