## 인증결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 테스트 → KG이니시스 → KG이니시스 선택 후 오른쪽 `+ 추가` 버튼 클릭

> 테스트 모드 설정시 포트원 테스트 계정으로 결제가 발생됩니다.

<Figure src={image1} caption="테스트 환경 설정 예시" />

4. 자동으로 입력된 값 확인 후 저장

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

<Hint style="info">
**KG이니시스 일반 결제용 PG상점아이디(MID) : `INIpayTest`**

**KG이니시스 정기 결제용 PG상점아이디(MID) : `INIBillTst`**

</Hint>

</Tab>

## 비 인증 결제

<Hint style="info">
- KG이니시스의 **API를 통한 빌링키 발급** 은 PG사를 통해 입점 가능 여부 확인이 우선적으로 필요합니다. 대부분의 경우 '**결제창을 이통한 빌링키 발급**' 방식으로 지원되며 'API를 통한 빌링키 발급' 방식은 이니시스 정책으로 인하여 입점이 까다로운 점 참고부탁드립니다.
- **해외 카드는 지원되지 않습니다**

</Hint>
