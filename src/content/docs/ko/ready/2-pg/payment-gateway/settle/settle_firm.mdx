## 관리자콘솔 설정 방법

<Tabs>
<Tab title="테스트 결제">
**내통장 결제는 **<mark style="color:red;">**테스트결제를 지원하지 않습니다**</mark>**.**

**반드시 실 계약이후 테스트 계정과 실계정을 발급받으셔야 테스트 연동이 가능합니다.**

</Tab>

<Tab title="실 결제">
**실 환경 구성방법**

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io)
2. 결제연동 탭 클릭
3. 결제 대행사 설정에 실연동 → 헥토파이낸셜 → 헥토파이낸셜 내통장결제 선택후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image5} caption="실 환경 설정 예시" />

4. 계약 후 발급받은 실상점 정보 기재후 저장
   - 암호화키는 헥토파이낸셜 상점 관리자의 <strong>가맹점정보 → 기본정보 → 라이센스 & 암호화키 정보 → 라이센스키(전문 해쉬 키)</strong>에서 열람 가능합니다.

<Figure src={image6} caption="암호화키 확인 방법" />

<Figure src={image7} caption="실 환경 상세 설정 예시" />

</Tab>
