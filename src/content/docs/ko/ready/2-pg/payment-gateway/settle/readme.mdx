## 인증결제 및 비인증결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io)
2. 결제연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 핵토파이낸셜 → 핵토파이낸셜 선택후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image1} caption="테스트 환경 설정예시" />

4. 자동입력된 정보 확인후 저장
   - **헥토파이낸셜 인증결제** 테스트 nxca_jt_il : 카드사 인증창 결제 요청
   - **헥토파이낸셜 비인증결제** 테스트 nxca_jt_bi : 카드번호, 유효기간 입력으로 결제 요청
   - **헥토파이낸셜 구인증결제** 테스트 nxca_ks_gu : 카드번호, 유효기간, 생년월일, 카드비밀번호 입력으로 결제 요청

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

</Tab>

<Tab title="실 결제">
#### **실** 환경 구성방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io)
2. 결제연동 탭 클릭
3. 결제 대행사 설정에 실연동 → 헥토파이낸셜 → 헥토파이낸셜 선택후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image3} caption="실 환경 설정 예시" />

4. 카드사 심사 완료 후 헥토파이낸셜에서 발급받은 실상점 정보 기재 후 저장

<Figure src={image4} caption="실 환경 상세 설정 예시" />

</Tab>
</Tabs>
