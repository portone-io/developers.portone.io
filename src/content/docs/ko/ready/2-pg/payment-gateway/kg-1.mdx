## 인증결제 및 비인증결제

<Tabs>
<Tab title="테스트 결제">
#### 테스트 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 테스트 → 모빌리언스 → 모빌리언스 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image1} caption="테스트 환경 설정 예시" />

4. 자동으로 입력된 값 확인 후 저장

<Figure src={image2} caption="테스트 환경 상세 설정 예시" />

</Tab>

<Tab title="실 결제">
#### 실 환경 구성 방법

1. [포트원 관리자 콘솔 로그인](https://admin.portone.io/integration)
2. 결제 연동 탭 클릭
3. 결제 대행사 설정에 실연동 → 모빌리언스 → 모빌리언스 선택 후 오른쪽 `+ 추가` 버튼 클릭

<Figure src={image3} caption="실 환경 설정 예시" />

4. 카드사 심사완료 후 모빌리언스에서 발급받은 실상점 정보 입력 후 저장

<Figure src={image4} caption="실 환경 상세 설정 예시" />

</Tab>
</Tabs>
