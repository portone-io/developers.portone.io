## 1. 포트원 회원가입 하기

<Hint style="info">
  결제창 연동을 진행하기 위해서는 포트원 관리자페이지 [**회원가입**](https://admin.portone.io)이 선행되어야 합니다.

  포트원 관리자 페이지는 **별도 계약 없이** **무료** 회원가입이 가능합니다.
</Hint>

<Details>
  <p slot="summary"><strong>포트원 회원가입 바로 하러 가기</strong></p>

  - 포트원 회원가입은 "**이메일 주소**"만 준비하시면 됩니다.
  - 비밀번호는 쉽게 유추할 수 없도록 "**복잡하게 설정하는 것**" 잊지 마세요.
</Details>

아래와 같은 화면에서 **가입하기**를 클릭하여 진행해 주시면 됩니다.

## 2. PG 정보 설정하기

포트원 회원가입 이후 관리자 페이지 내 **결제 연동** **->** **테스트 연동 관리** 탭에서 연동하고자 하는 PG사를 선택합니다.

**테스트 연동 관리** 메뉴에서는 포트원에서 지원하는 각 PG사별 테스트 설정을 진행할 수 있습니다.

<Figure src="/gitbook-assets/ko/image (74) (1).png" caption="결제연동 -> 테스트 연동 관리" />

<Hint style="info">
  각 PG사별 **테스트 설정 및 실 계정 설정 방법**이 **상이**합니다. 자세한 PG사별 설정 방법은 [**링크**](2-pg/readme)를 통해 확인하세요.
</Hint>

    ### Store Id 확인하기

    결제창 연동 및 API 호출 시 필요한 Store Id 정보를 확인하실 수 있습니다.

    1. [**포트원 관리자콘솔**](https://admin.portone.io) 로그인 -> **결제 연동** 탭 진입
    2. **Store Id** 확인

    <Figure src={getConsoleStoreIdImage} caption="포트원 관리자콘솔 로그인 -> 결제 연동 탭 -> Store Id 확인" />

    <Hint style="info">
      **Store Id 란?**

      - 고객사에서 포트원의 관리자 콘솔을 통해 생성한 **상점마다 고유하게 부여되는 값** 입니다.
      - 가장 처음 콘솔 계정을 만들게 되면 **대표 상점**이 생성되며, 추가로 여러 하위 상점을 생성할 수 있습니다.
      - 결제창 호출 시 `storeId` 파라미터에 해당 값을 넣어주셔야 합니다.
      - **기술 문의** 시 해당 값을 공유해 주시면 빠르게 문제 해결을 도와드릴 수 있습니다.
    </Hint>

    ### V2 API Secret 발급 받기

    V2 API를 연동하기 위해서는 V2 API Secret을 별도로 발급해야 하며, 아래 이메일로 문의하시면 발급해드리고 있습니다.

    > V2 기술지원 이메일: [tech.support@portone.io](mailto:tech.support@portone.io)

    <Hint style="info">
      **API Secret 이란?**

      - API 호출에 필요한 **Access Token**을 발급하기 위해 필요한 필수 값입니다.
      - V2 API Secret을 발급받으신 후, 이를 이용해 [API Secret을 이용한 토큰 발급](/api/rest-v2/auth#post%20%2Flogin%2Fapi-secret) API를 호출하시면 <br /> **Access Token**을 발급받으실 수 있습니다.
    </Hint>

    <Hint style="danger">
      API Secret은 **결제내역을 컨트롤**할 수 있는 **민감 정보**이기 때문에 외부로 유출되지 않도록
      유의하시기 바랍니다.
    </Hint>
